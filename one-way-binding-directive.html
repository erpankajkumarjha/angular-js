'@' : One way binding

One way binding means a parent sending anything to the directive scope through the attribute, gets reflected in the directive. But if any change in the directive happens it will not reflect in parent.

Syntax:

  

        var app= angular.module('applicaionName',[]);

        app.directive('directiveName', function(){
          return {
            restrict: 'EA',
            scope: {
              stringParam:'@'
            },
            template: "{stringParam}}"
          };
        });
